**textMe - Real-Time Private Chat Application**

textMe is a lightweight, real-time messaging platform built with Node.js and Socket.io. It features a clean Bootstrap UI and persistent data storage using a cloud-hosted MySQL database.

🚀 Features
Real-time Messaging: Instant message delivery using WebSockets.

Private Rooms: Chat privately with any online user.

Persistent Storage: Messages are saved in a MySQL database and retrieved upon login.

Responsive UI: Built with HTML5, CSS3, and Bootstrap 5.

Cloud Hosted: Backend deployed on Koyeb/Render with a Railway managed database.

🛠️ Tech Stack
Frontend: HTML, CSS (Bootstrap), JavaScript

Backend: Node.js, Express.js

Real-time Engine: Socket.io

Database: MySQL (Hosted on Railway)

Deployment: Koyeb / Render
